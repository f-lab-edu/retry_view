### 회원 가입/탈퇴 기능

### 리뷰 기능
제품 정보, 사용 후기, 별점, 작성자 정보 등을 제공

### 중고 물품 판매/구매 게시글 작성 기능
제품정보, 가격, 작성자 정보 제공
-> 작성된 게시글을 통해 채팅 가능 (+스크랩?)

### 사용자, 관리자 간 채팅 기능
관리자 -> 사용자 일방향 채팅
사용자 <-> 사용자 양방향 채팅
